Общие положения

Ограничения и допущения

  1. 1. Таблицы, содержащие составные первичные ключи не обрабатываются. К этой категории, например, относится группа таблиц с префиксом ‘RC$’ .
  2. 2. Анализируется только ссылочная целостность данных Oracle и “Технологического ядра”.
  3. 3. Имя схемы IBSO должно быть ‘IBS’. Иные имена не тестировались.
  4. 4. Тексты скриптов написаны для bash (включая порт под Cygwin).
  5. 5. Все оценки времени приведены для случая изготовления копии на персональном компьютере с двухядерным процессором 2.5 Ghz, 4Gb ОЗУ, IDE HDD.

Термины и определения

  • “ОРИГИНАЛ” - исходная схема
  • “КОПИЯ” - уменьшенный в объеме функциональный эквивалент “ОРИГИНАЛА”

Необходимые средства

  1. Oracle на платформу “КОПИИ”.
  2. Доступ к “ОРИГИНАЛУ” с правами владельца схемы IBS.
  3. Обновление для текущей версии “Технологического ядра”.
  4. Комплект скриптов minibso.zip
